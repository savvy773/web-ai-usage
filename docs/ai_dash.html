AI Usage Dashboard

로컬 Claude, Codex, Gemini CLI 사용량을 SvelteKit 서버에서 수집하고 JSON으로 저장한 뒤 브라우저 화면에 렌더링하는 대시보드입니다.

데이터 흐름

  1. 서버가 CLI를 node-pty 가상 터미널에서 실행합니다.
  2. Claude /usage, Codex /status, Gemini /model 출력을 수집합니다.
  3. parser가 TUI escape, cursor movement, ESC가 빠진 CSI 조각, carriage return, 그래프 문자를 정규화합니다.
  4. 정규화된 usage는 data/usage-history.json에 저장됩니다.
  5. /api/usage가 JSON payload를 반환하고 Svelte 화면이 이를 렌더링합니다.

파싱 안정화

  • \r\n은 줄바꿈으로 유지하고 단독 \r은 공백으로 처리합니다.
  • Gemini는 퍼센트 값이 채워진 마지막 Model usage redraw 화면을 우선합니다.
  • box, bracket, block/bar 그래프 문자는 공통 정규식으로 제거합니다.
  • Claude redraw로 남는 Curre t session, Rese s 형태도 current/reset으로 인식합니다.
  • Codex의 92% left 같은 표현은 dashboard 기준인 used percent로 변환합니다.

운영 메모

자세한 구현 구조와 API contract는 docs/architecture.md를 기준 문서로 관리합니다. 이 HTML은 브라우저에서 빠르게 열어보는 요약 문서입니다.
