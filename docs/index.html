Features

Everything you need
to track AI usage

For developers running multiple AI CLIs locally. No accounts, no API keys, no data leaving your machine.

⚡

Multi-Provider Collection

Collects Claude /usage, Codex /status, and Gemini /model via virtual terminals — no API keys needed.

↻

Smart Retry Logic

Up to 5 attempts per provider with phase diagnostics. Automatically re-enters slash commands if they disappear mid-session.

▦

Weekly Pace Card

Compare your actual usage bar against a minimum 20% threshold marker to stay on track through the week.

⏱

Reset Countdown

Live countdown to your next usage reset for each provider, so you always know how much runway is left.

📡

Live Server Logs

SSE-based real-time log panel via /api/server/logs — streams directly to the browser, no polling.

⬡

Dual Cache

Server-side JSON history in 10-minute buckets, plus browser localStorage fallback for instant first render.

Quick Start

Up in one command

Clone, install, and run. The dashboard opens at 127.0.0.1:5173.

one-liner install · powershell 
irm https://raw.githubusercontent.com/savvy773/ai_usage/main/scripts/install.ps1 | iex
manual install 
# clone & install
git clone https://github.com/savvy773/ai_usage.git
cd ai_usage
pnpm install

# start the dashboard
.\scripts\start-server.ps1 -Open
options 
.\scripts\start-server.ps1 -Open
  # open browser automatically

.\scripts\start-server.ps1 -Port 5173
  # use custom port

.\scripts\start-server.ps1 -Mode preview
  # production preview build

.\scripts\start-server.ps1 -Status
  # show server status
