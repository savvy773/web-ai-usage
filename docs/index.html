Features

Everything you need
to track AI usage

Designed for developers running multiple AI CLIs locally. No accounts, no data leaving your machine.

⚡

Multi-Provider Collection

Collects Claude /usage, Codex /status, and Gemini CLI /model via virtual terminals — no API keys needed.

↻

Smart Retry Logic

Up to 5 attempts per provider with phase diagnostics. Automatically re-enters slash commands if they disappear mid-session.

📊

Weekly Pace Card

Compare your actual usage bar against a minimum 20% threshold marker to stay on track through the week.

⏱

Reset Countdown

Live countdown to your next usage reset for each provider, so you always know how much runway is left.

📡

Live Server Logs

SSE-based real-time log panel powered by /api/server/logs — no polling, no page reload.

💾

Dual Cache

Server-side JSON history with 10-minute buckets, plus browser localStorage fallback for instant first render.

Quick Start

Up in one command

Clone the repo and run the start script. The dashboard opens at http://127.0.0.1:5173.

PowerShell 
# clone & install
git clone https://github.com/savvy773/ai_usage.git
cd ai_usage
pnpm install

# start the dashboard
.\scripts\start-server.ps1 -Open
Options 
.\scripts\start-server.ps1 -Open         # open browser
.\scripts\start-server.ps1 -Port 5173   # custom port
.\scripts\start-server.ps1 -Mode preview # production preview
.\scripts\start-server.ps1 -Status       # server status
